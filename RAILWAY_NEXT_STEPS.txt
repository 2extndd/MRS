Автоматическое создание сервисов через API НЕ РАБОТАЕТ.

Ошибки API:
• "Not Authorized" при создании PostgreSQL
• "Problem processing request" при создании сервисов

Решение: Ручная настройка через Railway Dashboard (3 минуты)

1. Откройте Railway Dashboard:
   https://railway.app/project/f17da572-14c9-47b5-a9f1-1b6d5b6dea2d

2. Следуйте инструкции:
   QUICKSTART.md (быстрый старт)
   или
   SETUP_INSTRUCTIONS.md (подробная инструкция)

3. Действия:
   • Добавить PostgreSQL database
   • Подключить GitHub repo (2extndd/MRS)
   • Создать WEB сервис
   • Создать WORKER сервис
   • Скопировать переменные окружения (см. выше)
   • Включить публичный домен для WEB
